### Installation

This system requires [Node.js](https://nodejs.org/) and [Sails.js](https://sailsjs.com) to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd sails.js-EventManagementSystem
$ npm i
$ sails lift
```
